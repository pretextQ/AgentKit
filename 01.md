# 智能OnCall Agent系统

## 项目介绍

基于 LangChain 构建的**智能 Agent 系统**，整合 **RAG 知识库问答**、**ReAct 对话**和 **Plan-Execute 故障诊断**三大核心能力，实现问题自动应答与故障智能排查。

**技术栈：** LangChain、LangGraph、RAG、ReAct、Plan-Execute-Replan、MCP

## 技术亮点

- **多 Agent 架构：** 基于 **LangGraph 图编排**实现模块化 Agent 工作流，通过 **MCP 协议**集成外部工具（如数据库查询、API 调用等），使 Agent 能够灵活调用工具完成复杂任务。

- **RAG 知识库系统：** 设计完整的**文档向量化存储与检索方案**，针对分块大小与检索 **TopK 参数调优**，解决"分块过小丢失上下文、过大降低精度"的平衡问题，知识检索准确率达到 **85%+**。

- **流式对话与上下文记忆：** 基于 **ReAct 模式**实现对话 Agent，支持多场景无缝切换；基于 **SSE** 实现流式输出，配合 **MemorySaver** 实现多轮对话上下文记忆。

- **Plan-Execute-Replan 工作流：** 实现**自动规划 → 工具调用 → 结果分析 → 动态调整**的完整闭环，能够根据输入自动检索知识库、调用工具查询数据、分析结果并生成建议。
